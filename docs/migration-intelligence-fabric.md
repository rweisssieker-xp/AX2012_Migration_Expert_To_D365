# Migration Intelligence Fabric

Migration Intelligence Fabric is now implemented as the reusable intelligence layer above analysis, governance, Commerce, testing, cutover, and solo orchestration. It converts project evidence into memory, benchmarks, portfolio control, scenario simulations, quality audits, war games, and value tracking.

## Implemented Scope

| Area | Runtime coverage |
| --- | --- |
| Feature range | Features 381-500 added to the AI/KI USP list. |
| Skills | 20 intelligence skills, including memory, benchmarking, portfolio, scenario, quality, war game, and value realization specialists. |
| Templates | 61 intelligence templates for scorecards, ledgers, maps, audits, runbooks, and executive briefs. |
| Configs | 10 JSON rule files for benchmarking, portfolio, scenario, quality, technical debt, Fabric, integration resilience, security, and value rules. |
| CLI commands | 16 commands from `intelligence-pack` through `continuous-improvement`. |
| Validator | Checks counts, feature continuity, docs coverage, template mapping, command docs, and intelligence smoke outputs. |

## Automation Boundary

The plugin can generate evidence packs, readiness results, action plans, commands, and gate recommendations. Real executive, CISO, legal, audit, payment/PCI, finance, and production approvals remain external sign-offs and are represented as evidence gates.
