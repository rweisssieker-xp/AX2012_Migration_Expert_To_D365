# Project Operating Model

## Governance Cadence
| Meeting | Frequenz | Teilnehmer | Input | Output |
| --- | --- | --- | --- | --- |
| Daily Delivery Sync | Taeglich | Workstream Leads | Tasks, Blocker | Tagesprioritaeten |
| Migration Factory Review | Woechentlich | PM, Leads | Analyzer Reports | Wellen- und Sprintentscheidungen |
| Security Gate | Nach Gate | CISO, Security, Architecture | CISO Pack | Freigabe / Auflagen |
| Steering Committee | 2-woechentlich | CEO, CIO, CISO, PM | Steering Pack | Entscheidungen |

## Working Agreements
- Analyzer ist Fuehrungssystem fuer Scope-Transparenz.
- Jede Entscheidung bekommt Owner, Datum und Evidence.
- Jedes Risiko bekommt Mitigation und naechste Aktion.
- Jede Welle braucht Gate-Kriterien und Rollback-Plan.
