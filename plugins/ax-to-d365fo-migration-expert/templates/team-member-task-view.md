# Team Member Task View

## Zweck
Technische und fachliche Arbeitspakete werden automatisch aus Analysefunden in konkrete Aufgaben uebersetzt.

## Aufgabenstruktur
| Objekt | Rolle | Aufgabe | Akzeptanzkriterium | Evidence |
| --- | --- | --- | --- | --- |
|  | Developer | Code pruefen/refactoren | Build und Tests bestanden | PR Link |
|  | Functional Consultant | Standardersatz validieren | Prozessabnahme dokumentiert | Fit-gap Result |
|  | Data Lead | Datenqualitaet klaeren | Bereinigungsregel definiert | Profiling Report |
|  | Security Lead | Rollen/SOD pruefen | Freigabe erteilt | Security Gate |

## AI-KI-USP
Die AI-KI erzeugt sprintfaehige Arbeit statt langer Migrationslisten.
