## Disposition Rules

| Disposition | Use when |
| --- | --- |
| Retire | No active business value or usage evidence exists. |
| Standard | D365FO standard functionality replaces the customization. |
| Configure | Setup or feature management satisfies the need. |
| Extend | Extension, Chain of Command, or event handlers can preserve behavior. |
| Rebuild | The AX pattern is incompatible and must be redesigned. |
| ISV | A maintained D365FO ISV solution is a better target. |
