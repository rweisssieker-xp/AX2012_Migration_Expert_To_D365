# Project Manager Control View

## Steuerungsziel
Automatisch generierte Projektsteuerung fuer Umfang, Risiken, Abhaengigkeiten, Ressourcen, Gate Readiness und Eskalationen.

## Projekt-Cockpit
| Dimension | Automatische Quelle | Aktion |
| --- | --- | --- |
| Scope | Inventory, Disposition, Do-not-migrate | Scope Freeze / Change Request |
| Aufwand | Effort Estimate, Complexity | Sprint- und Ressourcenplanung |
| Risiko | Heatmap, RAID, Security View | Eskalation / Mitigation |
| Abhaengigkeiten | Dependency Graph, Wave Roadmap | Termin- und Cutoverplanung |
| Qualitaet | Quality Gates, Evidence Confidence | Gate-Freigabe |

## Automatisch erzeugbare PM-Artefakte
- RAID Log
- RACI Matrix
- Weekly Status Report
- Steering Committee Pack
- Entscheidungs- und Eskalationsliste
