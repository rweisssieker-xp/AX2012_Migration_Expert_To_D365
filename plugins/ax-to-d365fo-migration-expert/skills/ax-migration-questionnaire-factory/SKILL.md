# AX Migration Questionnaire Factory

## Purpose

Create guided workshop and operating-model packs for AX to D365FO programs.

## Command

```powershell
python axmigrate.py questionnaire --persona all --output migration-questionnaires/sample
```

## Outputs

- CEO, CIO, CISO, PMO, team, and change questionnaires.
- `migration-factory-mode.md`
- `cutover-war-room-live-pack.md`
- `hypercare-command-center.md`
- `partner-sales-accelerator.md`
- `questionnaire-schema.json`

## Quality Bar

Convert answers into decisions, risks, backlog items, evidence gaps, and persona pack updates.
