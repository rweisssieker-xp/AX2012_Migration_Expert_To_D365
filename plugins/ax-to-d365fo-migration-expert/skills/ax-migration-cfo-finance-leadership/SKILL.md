# AX Migration CFO Finance Leadership

## Purpose

Translate migration evidence into finance leadership decisions: budget, benefits, TCO, audit, controls, and closing readiness.

## Outputs

- Budget control pack.
- ROI and benefits realization.
- TCO model.
- Audit and closing readiness.
- Financial decision log.

## Command

```powershell
python axmigrate.py stakeholder-pack migration-analysis/sample --stakeholder cfo --output stakeholder-packs/cfo
```

## Quality Bar

Use ranges and assumptions. Separate one-time migration cost, run cost, avoided effort, compliance risk, and benefits realization.
