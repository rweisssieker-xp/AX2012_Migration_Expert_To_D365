# AX Migration Partner Sales

## Purpose

Turn the plugin into a repeatable consulting and sales accelerator.

## Outputs

- Assessment offer generator.
- Proposal/SOW generator.
- Pricing assumption pack.
- Client executive pitch pack.

```powershell
python axmigrate.py stakeholder-pack migration-analysis/sample --stakeholder partner-sales --output stakeholder-packs/partner-sales
```
