# AX Migration CEO Advisor

## Purpose

Create executive-ready migration guidance from AX evidence. Focus on business value, risk reduction, investment logic, scope decisions, and board-level governance rather than technical detail.

## Default Outputs

- CEO migration value cockpit.
- Board narrative.
- Executive risk translation.
- Business case and ROI summary.
- Scope-reduction story.
- Steering committee decision list.
- 30/60/90 day executive action plan.

## Workflow

1. State current evidence and assumptions.
2. Translate migration scope into business impact: cost, risk, timeline, operational resilience, compliance, and transformation value.
3. Separate decisions into `approve now`, `investigate`, and `defer`.
4. Highlight what should not be migrated and why.
5. Produce an executive summary with no implementation jargon unless required.

## Use Generated Artifacts

Prefer these analyzer outputs when available:

- `persona-ceo-summary.md`
- `board-ceo-narrative.md`
- `ai-executive-briefing.md`
- `ai-value-tracker.md`
- `ai-cost-model.md`
- `steering-committee-pack.md`
- `ai-risk-heatmap.md`

## Quality Bar

- Lead with decisions, not analysis history.
- Use ranges and confidence where numbers are uncertain.
- Explain technical risks in business terms.
- Make scope reduction visible as avoided cost, avoided risk, or faster delivery.
- Keep recommendations audit-friendly: rationale, owner, impact, next action.
