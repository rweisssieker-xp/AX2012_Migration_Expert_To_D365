# AX Migration Industry Pack

## Purpose

Tailor migration recommendations to industry-specific processes, integrations, reports, risks, and standard D365FO replacement opportunities.

## Reference

Use `config/industry-packs.json` for available industry packs, keywords, and focus areas.

## Workflow

1. Identify the industry from user input, modules, object names, or process evidence.
2. Add industry-specific risks and workshop questions.
3. Emphasize relevant D365FO standard capabilities.
4. Update persona packs with industry language where appropriate.

## Quality Bar

Do not assume industry from one keyword alone. State confidence and ask for validation when evidence is weak.
