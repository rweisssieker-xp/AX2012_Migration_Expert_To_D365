# AX Migration Enterprise Architect

## Purpose

Connect AX migration decisions to enterprise architecture, capability mapping, portfolio rationalization, target landscape, and technical debt reduction.

## Outputs

- Capability map.
- Application portfolio impact.
- Technical debt burn-down.
- Target landscape blueprint.
- Dependency risk graph.

## Command

```powershell
python axmigrate.py stakeholder-pack migration-analysis/sample --stakeholder enterprise-architect --output stakeholder-packs/enterprise-architect
```

## Quality Bar

Separate business capability, application component, integration, data domain, technical debt, and governance decision.
