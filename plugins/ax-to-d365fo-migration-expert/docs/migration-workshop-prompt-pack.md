# AI Prompt Pack for Migration Workshops

## Finance Fit-Gap

Analyze these AX finance processes and identify D365FO standard fit, gaps, risks, and required decisions.

## SCM Fit-Gap

Review procurement, sales, inventory, warehouse, and manufacturing evidence and produce a fit-gap matrix.

## Integration Workshop

Classify each integration by current technology, target D365FO pattern, SLA, monitoring, replay, and owner.

## Data Migration Workshop

Separate configuration, master, open transactions, history, attachments, and archive scope. Define reconciliation.

## Report Rationalization

Classify reports as retire, standard, SSRS, Power BI, Financial Reporter, or archive.

## Cutover Rehearsal

Build a timed cutover runbook with dependencies, validation points, rollback criteria, and communications.

## Executive Steering

Summarize complexity, risk, budget drivers, scope reduction, decisions needed, and next 30/60/90 days.
